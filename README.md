# Custom Reaction Clipboard

A sleek, modern, and lightweight desktop application built with Python and PyQt6. It allows you to create a custom, searchable clipboard for your favorite meme and reaction images (`.png`, `.jpg`, etc.) and instantly copy them to your system clipboard to paste into Discord, Slack, or any other app.

##  Features
* **Cross-Platform:** Works flawlessly on Linux and Windows.
* **Modern UI:** A beautiful dark-mode aesthetic with rounded corners and hover effects.
* **In-App Uploads:** Easily add and name new reactions directly through the UI.
* **Instant Copy:** Click any reaction tile to instantly copy the image to your clipboard.
* **Lightning Fast Search:** Filter through your custom reactions in real-time.

## 🚀 Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/abhinavkollery/imagei-clipboard.git
   cd imagei-clipboard
   ```

2. Install the required dependencies:
   ```bash
   pip install PyQt6
   ```

3. Run the application:
   ```bash
   python3 main.py
   ```

## 🛠️ Usage
* **Adding Reactions:** Click the `+ Add Reaction` button, select an image from your computer, and give it a search name. The app automatically copies the image to the local folder and updates the database.
* **Copying:** Simply click on any image tile. The app will minimize, and the image is now on your clipboard ready to be pasted!
